Conociendo algunas etiquetas y sus usos

Vamos a ver en vivo como funciona algunas de las etiquetas como quedarían dentro de mi web

  1. La etiqueta p se utiliza para definir un párrafo
  2. La etiqueta br me inserta un salto d linea
  3. La etiqueta hr me inserta una linea divisoria
  4. La etiqueta img me inserta una imagen
  5. La etiqueta div me crea secciones
